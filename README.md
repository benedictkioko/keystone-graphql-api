# keystone-graphql-api

Headless CMS &amp; GraphQL API with KeystoneJS

This project contains an AdminUI, GraphQL PlayGround and GraphQL App.

## Running the Project.

To run this project first run `yarn` to install the project packages.

```

    yarn
```
